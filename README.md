# Edlyn Media - Vite + React (Custom)

Features:
- Centered content with max-width 1200px
- Banner with fade overlays (Option A). CSS for Option C is included in comments.
- Videos and Shorts loaded from `src/videos.json` (local JSON).
- Modal player for playing videos.

How to run:
1. npm install
2. npm run dev

To add more videos/shorts: edit `src/videos.json` and add objects with `{ "id": "<youtube id>", "title": "..." }`
